Steps to Install: 
 
1) Install python -  
a) Download installer and follow instructions - https://www.python.org/ftp/python/2.7.14/python-2.7.14.amd64.msi 
 
b) C:\Python27\Tools\Scripts\win_add2path.py -> Run this command on the terminal. This will set the path to access python on the terminal. Restart system 
 
c) python --version -> This will display the python version installed. Python 2.7.14 
 
2) Install ipython - Interactive python notebook 
           a) pip install "ipython[notebook]"  - Run this command on terminal 
 
3) pip install numpy pandas scipy sklearn plotly seaborn – Run this command on the terminal 
 
4) git clone "https://github.com/supr1921/ML-Brownbag"

5) Navigate to the root dir and run - ipython notebook
